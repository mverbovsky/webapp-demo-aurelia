# Představení
Aurelia je UI framework postavený na ES 2015 a ES 2016. Nejedná se o monolitický framework, ale o soubor samostaných knihoven (modulární framework), které lze používat podle potřeby. Framework je primárně určený pro psaní SPAs (single page application).
